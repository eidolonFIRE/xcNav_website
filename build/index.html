Welcome to xcNav

A free and open-source paramotor app.

Features

Group Coordination

  • Live location sharing
  • Syncronized flight plan
  • Text Chat (with text-to-speech)

Tools and Instruments

  • Audio cues
  • ADSB-in
  • Airspace overlay
  • Ground elevation
  • Advanced wind indicator
  • Log book with in-depth stats
  • Fuel usage calculations
